**CALCULATOR_PROJECT.**
A simple and responsive Calculator built using HTML, CSS, and JavaScript.
It performs basic arithmetic operations and supports scientific functions with a clean UI.

___Basic Operations__
1.Addition
2.Subtraction
3.Multiplication
4.Division

__Scientific Functions__
1.sin(), cos(), tan()
2.log(), ln()
3.Square (x²)
4.Square root (√x)

__Tech Stack__
1.HTML5 – Structure of the calculator
2.CSS3 – Styling & layout
3.JavaScript (ES6) – Logic & functionalit
